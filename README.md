# Voice Dictation ROI Calculator

Calculate the time and money your team could save by adopting voice dictation.

**[Try it live](https://swv-roi-calculator.netlify.app)**

## What it does

An interactive calculator that models the return on investment of switching from typing to voice dictation across a team.  Input your team size, typing habits, and costs -- get back annual savings, ROI percentage, breakeven timeline, and a 3-year projection.

Includes presets for solo professionals, teams, and company-wide evaluation, plus advanced controls for adoption rate, accuracy overhead, and proofreading time.

## Built with

React, TypeScript, Tailwind CSS, Recharts, Framer Motion.  Client-side PDF report generation via `@react-pdf/renderer`.

Part of [WithVoice](https://withvoice.ai) -- AI fluency for teams.
